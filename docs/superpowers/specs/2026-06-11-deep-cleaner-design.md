# 深度清理工具 - 设计文档

## 1. 项目概述

### 1.1 项目名称
深度清理工具 (Deep Cleaner)

### 1.2 项目目标
创建一个功能全面、界面优雅的Windows系统清理工具，帮助用户深度扫描、分析和清理电脑中的各类垃圾文件，释放磁盘空间，优化系统性能。

### 1.3 目标用户
- 需要清理电脑垃圾文件的普通用户
- 需要分析磁盘空间的专业用户
- 需要管理启动项和软件的系统管理员

## 2. 技术架构

### 2.1 技术栈
- **前端框架**：Electron + React 18 + TypeScript
- **UI框架**：Tailwind CSS 3.x
- **动画库**：Framer Motion 10.x
- **图表库**：Recharts 2.x
- **状态管理**：Zustand 4.x
- **打包工具**：Electron Builder
- **构建工具**：Vite
- **包管理器**：npm 或 yarn

### 2.2 项目结构

## 3. 功能设计

### 3.1 首页仪表盘

#### 3.1.1 存储概览
- 显示所有磁盘驱动器的使用情况
- 环形图展示各盘容量占比
- 显示总容量、已用空间、可用空间
- 预估可清理空间

#### 3.1.2 扫描统计
- 文件类型分布柱状图（按大小排序）
- 缓存/临时文件占比
- 安装包/日志/重复文件等分类统计

#### 3.1.3 自定义快捷功能
- 用户可自选最多4个常用功能放在首页
- 可选功能包括：深度扫描、大文件查找、重复文件查找、注册表清理、软件管理、启动项管理、清理计划、清理历史
- 拖拽排序支持

### 3.2 磁盘扫描功能

#### 3.2.1 扫描范围
- 用户自定义选择要扫描的盘符（C、D、E等）
- 支持扫描系统目录、用户目录、程序目录
- 可选扫描特定文件夹

#### 3.2.2 扫描模式
- **快速扫描**：扫描常见垃圾文件位置
- **深度扫描**：扫描所有文件，全面分析
- **自定义扫描**：用户指定扫描路径和文件类型

#### 3.2.3 扫描进度展示
- 实时文件路径滚动显示
- 进度条显示整体进度
- 已扫描文件数量统计
- 发现可清理项统计
- 预估剩余时间
- 扫描动画效果

#### 3.2.4 多线程并行扫描
- 使用Node.js Worker Threads
- 同时扫描多个目录
- 充分利用多核CPU
- 支持暂停/继续扫描

### 3.3 文件分类系统

#### 3.3.1 系统缓存
- Windows临时文件（%TEMP%、Windows\Temp）
- 浏览器缓存（Chrome、Edge、Firefox等）
- 系统更新缓存
- 图标缓存
- 缩略图缓存
- DNS缓存
- Windows日志文件

#### 3.3.2 临时文件
- 用户临时文件夹内容
- 程序临时文件
- 下载文件夹中的临时文件
- 回收站文件

#### 3.3.3 安装包
- 已下载的安装程序（.exe、.msi）
- 驱动程序备份
- Windows更新包
- 应用程序安装缓存

#### 3.3.4 日志文件
- Windows事件日志
- 应用程序日志
- 安装日志
- 崩溃报告

#### 3.3.5 重复文件
- 基于MD5/SHA256哈希比对
- 支持按文件名、大小、内容比对
- 智能保留建议（保留最新/最大/指定位置）

#### 3.3.6 大文件分析
- 可视化大文件分布（树状图）
- 按大小排序
- 按类型分组
- 按修改时间筛选

#### 3.3.7 空文件夹
- 扫描空目录
- 识别无效的文件夹结构
- 安全删除建议

#### 3.3.8 其他可清理项
- 旧的Windows安装（Windows.old）
- 系统还原点（保留最近N个）
- 休眠文件（可选禁用）
- 页面文件（可选调整）

### 3.4 清理策略

#### 3.4.1 风险等级标识
- **安全（绿色）**：确定可安全删除，如临时文件、缓存
- **中等（黄色）**：可能需要，如某些日志、旧安装包
- **危险（红色）**：重要文件，删除可能影响系统或程序

#### 3.4.2 智能建议
- 自动勾选安全项
- 根据文件年龄、使用频率给出建议
- 显示删除后预估释放空间

#### 3.4.3 删除模式
- **回收站模式**：默认移到回收站，可恢复
- **永久删除**：直接删除，不可恢复
- 安全文件默认直接删除
- 中等/危险文件默认回收站模式
- 用户可全局设置或逐项设置

#### 3.4.4 批量操作
- 按分类批量选择/取消
- 全选/全不选
- 自定义批量规则

### 3.5 高级功能

#### 3.5.1 注册表清理
- 无效的文件关联
- 无效的程序路径
- 无效的启动项
- 无效的COM/ActiveX组件
- 备份注册表后再清理

#### 3.5.2 软件管理
- 已安装软件列表
- 软件大小、安装日期、使用频率
- 一键卸载
- 残留文件清理

#### 3.5.3 启动项管理
- 开机启动项列表
- 启用/禁用启动项
- 启动项来源分析
- 启动影响评估

#### 3.5.4 系统优化建议
- 磁盘碎片整理建议
- 系统服务优化建议
- 视觉效果优化建议
- 电源计划优化建议

### 3.6 智能功能

#### 3.6.1 清理计划
- 定时自动清理（每日/每周/每月）
- 自定义清理规则
- 静默清理模式
- 清理前确认选项

#### 3.6.2 清理历史
- 清理记录（时间、清理项、释放空间）
- 清理效果趋势图
- 导出清理报告
- 清理前后对比

#### 3.6.3 智能推荐
- 根据使用习惯推荐清理项
- 磁盘空间不足时提醒
- 定期清理提醒
- 大文件下载后提醒

## 5. 数据流设计

### 5.1 状态管理
使用Zustand管理全局状态：
- `useScanStore`：扫描状态、进度、结果
- `useCleanStore`：清理选项、历史记录
- `useSettingsStore`：用户设置、偏好
- `useDashboardStore`：仪表盘数据、快捷功能

### 5.2 IPC通信
- 主进程负责文件系统操作、系统API调用
- 渲染进程负责UI展示、用户交互
- 通过IPC进行双向通信
- 使用TypeScript类型定义确保类型安全

### 5.3 数据持久化
- 用户设置：存储在用户目录下的JSON文件
- 清理历史：SQLite数据库
- 扫描缓存：内存缓存，关闭后清除

## 7. 安全考虑

### 7.1 系统保护
- 不删除系统关键文件
- 重要操作前创建还原点
- 回收站保护机制

### 7.2 数据保护
- 不扫描/删除用户文档（除非明确指定）
- 敏感文件二次确认
- 删除前预览

### 7.3 注册表保护
- 修改前自动备份
- 提供恢复功能
- 仅清理确定无效的项
